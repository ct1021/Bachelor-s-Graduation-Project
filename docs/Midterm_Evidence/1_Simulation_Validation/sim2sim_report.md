# G1 RL 泛化性验证：跨引擎 Sim2Sim 域随机化 (Domain Randomization) 实验报告

## 1. 实验目标与背景
本节实验旨在验证本组基于 MuJoCo 仿真环境所训练的 PPO + 行为克隆 (BC) 混合目标策略，在面对与训练集不同的物理参数时，是否具备足够的**鲁棒性 (Robustness)** 与**泛化能力 (Generalization)**。由于真实物理世界 (Real World) 中往往存在不可避免的模型误差与动态干扰，因此在向物理硬件部署 (Sim2Real) 前，利用**域随机化 (Domain Randomization, DR)** 进行验证是必不可少的环节。

## 2. 实验设置
本次验证依托 `biped_rl` 环境与自定义拓展的 `G1DomainRandomizationEnv` 评价环境。在每轮评估重置 (Reset) 时，动态向环境注入如下两项核心物理维度的随机扰动：
1. **质量摄动 (Mass Randomization)**: 对 G1 机器人主体所有的连杆 (links) 注入从基准值向上向下按特定百分比浮动的随机质量。
2. **摩擦力摄动 (Friction Randomization)**: 对地面及机器人的碰撞体接触面摩擦系数进行同等幅度的随机扰动。

为全面评估算法边界，设置了以下四组噪声等级 (Noise Levels)：
- **基线 (Baseline)**: ±0% 扰动
- **轻度干扰 (Light)**: ±10% 扰动
- **中度干扰 (Moderate)**: ±20% 扰动
- **重度干扰 (Severe)**: ±30% 扰动

在每组噪声下运行 10 个 Episode，最高测试步数为 1000 步。

## 3. 结果分析
利用 `scripts/plot_sim2sim_robustness.py` 对 50 万步预训练策略 (`g1_ppo_500k.zip`) 进行了评估。结果如下：

![Sim2Sim 鲁棒性验证柱状图](./figures/sim2sim_robustness.png)

1. **零干扰基线表现极佳**：由于直接源自动作捕捉数据的冷启动，策略在无物理摄动时可以几乎完美完成 1000 步的稳定行走。
2. **中等干扰下的强鲁棒性**：在注入 ±10% 与 ±20% 级的高动态物理参数漂移时，由于 PPO 在探索中形成的网络容错能力，G1 大部分情况下仍能重新调整姿态存活，无明显“一步摔机”的情况。
3. **策略衰减临界点**：当噪声注入达到 ±30% 时，模型面临了极大的质量偏移及脚底打滑挑战，导致平均生存步数出现下降。这符合预期，也为下半学期的真机下发奠定了坚实的验证基础和底线区间。

## 4. 结论与下一步计划
本项实验已证明当前采取的混合奖励算法不仅仅过拟合于理想化的仿真引擎（例如仅在特定的 MuJoCo 参数下才收敛），而是**具备走向真实场景的泛化潜力**。
